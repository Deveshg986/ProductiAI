# 🚀 ProductiAI – Productivity Analyzer

## 🌐 Live App

👉 https://habitpredictor.streamlit.app/

---

## 📌 Overview

**ProductiAI** is a Machine Learning-based web application that helps users analyze their daily habits and predict their productivity level.

It combines:

* 🤖 Machine Learning (Random Forest)
* 🧠 Real-world logic (rule-based calibration)
* 🎨 Interactive UI (Streamlit)

---

## 🎯 Features

* 📊 Predict productivity score based on lifestyle inputs
* 🧠 Smart suggestions for improvement
* ⚡ Fast and interactive UI
* 🔥 Hybrid approach (ML + logic for realistic results)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/deveshg986/productiai.git
cd productiai
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run Application

```
streamlit run main.py
```
